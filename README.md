# SysML and KerML standard libraries

This is a **modified version** of the KerML and SysML v2 normative model libraries originally published in the [SysML-v2-Release](https://github.com/Systems-Modeling/SysML-v2-Release) repository (commit 834a3daf073b1886244833d9341677eb9733d1e4, January 8th, 2026). These modifications are documented in the CHANGES.md file in each library folder.

The original libraries are licensed under the GNU Lesser General Public License v3.0 (LGPLv3). This modified version is distributed under the same license. The original licenses (LICENSE, LICENSE-GPL) and the vendor's README (VENDOR-README.md) are included in every distribution.

## Libraries

- **stdlib** — The standard library, restructured and with select bug fixes. See `stdlib/CHANGES.md` for details.
- **redlib** — An extremely reduced version of the stdlib, primarily useful for basic modeling and for authors testing tooling gradually. See `redlib/CHANGES.md` for details.
